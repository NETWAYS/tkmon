NAME

Alert::Handler::Email - Parse and modify email messages.

VERSION

This document describes Alert::Handler::Email version 0.0.1

SYNOPSIS

Example 

	use Alert::Handler::Email;
	my $email = parseEmailStr($email_str);
	my $subject = getSubject($email);
	my $modifiedEmail = replaceSubject($email,'TK-Monitoring modified Subject');
	my $subject = getSubject($modifiedEmail);
	my $body = getBody($email);

DESCRIPTION

Alert::Handler::Email parses RFC2822 email messages. The parsing methods work with msg objects, in order to get such an object the parseEmailStr method has to be called. This method takes an email as string as an argument an returns a msg object.

METHODS

parseEmailStr

Example: 

	my $email = parseEmailStr($email_str);

Parses $email_str, a email message as string. Returns an Email::Simple msg object for further usage.

getSubject

Example: 

	my $subject = getSubject($email);

Returns the subject of the given msg object $email - $email is an object returned by parseEmailStr.

getBody

Example: 

	my $body = getBody($email);

Returns the body of the given msg object $email - $email is an object returned by parseEmailStr.

duplicateEmail

Example: 

	my $dupEmail = duplicateEmail($email);

Duplicates/Copies an email (Email::Simple) into a new Email::Simple object.

replaceSubject

Example: 

	replaceSubject($email,'TK-Monitoring modified Subject');

Replaces the subject of the given msg object $email with the second parameter. $email is an object returned by parseEmailStr.

replaceBody

Example: 

	replaceBody($email,'TK-Monitoring modified Body');

Replaces the body of the given msg object $email with the second parameter. $email is an object returned by parseEmailStr.

toString

Example: 

	print toString($modifiedEmail);

Returns the msg object as string.

DIAGNOSTICS

Cannot parse an empty email message string.
The given email string to parseEmailStr is empty.
Cannot parse an undefined msg.
The email msg object is undefined.

CONFIGURATION AND ENVIRONMENT

Alert::Handler::Email requires no configuration files or environment variables.
